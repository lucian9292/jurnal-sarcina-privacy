Jurnal de sarcina

Politica de Confidentialitate

1. Ce date colectam

Date de cont: adresa de email pentru autentificare prin Firebase.

Date de sarcina: data ultimei menstruatii (DUM), prenumele tau si al bebelusului, judetul, grupa sanguina Rh.

Date de sanatate: greutate, tensiune arteriala, simptome zilnice, miscari fetale, contractii.

Jurnal: notite text, fotografii (stocate local pe dispozitiv), starea de spirit.

2. Cum folosim datele

Datele sunt folosite exclusiv pentru functiile aplicatiei: calcularea saptamanii de sarcina, calendarul de investigatii, notificari locale, sugestii clinici, export PDF. Nu vindem si nu partajam datele cu terti in scop publicitar.

3. Unde sunt stocate datele

Local pe dispozitiv (AsyncStorage) pentru functionare offline. Firebase Firestore (Google Cloud) pentru sincronizare cloud. Firebase respecta GDPR si este certificat ISO 27001.

4. Fotografii

Fotografiile adaugate in jurnal sunt stocate exclusiv local pe dispozitiv. Nu sunt incarcate pe niciun server.

5. Drepturile tale (GDPR)

  • Acces: toate datele sunt vizibile in aplicatie
  • Corectare: le poti edita din sectiunea Profil
  • Stergere: stergerea contului elimina toate datele din cloud
  • Portabilitate: export prin functia Export PDF

6. Servicii terte

Firebase Authentication si Firestore (Google). Nu folosim reclame sau trackere de terta parte.
